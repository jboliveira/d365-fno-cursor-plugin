# Artifact Extensibility Guides

Condensed pointers to Microsoft artifact-specific guidance:

1. [Extensible classes](https://learn.microsoft.com/dynamics365/fin-ops-core/dev-itpro/extensibility/extensible-classes) — clear purpose, minimal public surface
2. [Extensible methods](https://learn.microsoft.com/dynamics365/fin-ops-core/dev-itpro/extensibility/extensible-methods) — extract switch/while/if blocks
3. [Extensible forms](https://learn.microsoft.com/dynamics365/fin-ops-core/dev-itpro/extensibility/extensible-forms) — extension-friendly form design
4. [Extensible EDTs](https://learn.microsoft.com/dynamics365/fin-ops-core/dev-itpro/extensibility/extensible-edts) — extend base types correctly
5. [Extensible enums](https://learn.microsoft.com/dynamics365/fin-ops-core/dev-itpro/extensibility/extensible-enums) — use extensible base enums
6. [Extensible delegates](https://learn.microsoft.com/dynamics365/fin-ops-core/dev-itpro/extensibility/extensible-code-delegates) — design multicast extension points
7. [Extensible tables](https://learn.microsoft.com/dynamics365/fin-ops-core/dev-itpro/extensibility/extensible-tables) — table extension patterns
8. [Extensibility attributes](https://learn.microsoft.com/dynamics365/fin-ops-core/dev-itpro/extensibility/extensibility-attributes) — Replaceable, Wrappable, ExtensionOf
